##  기능 요구 사항

하노이의 탑 게임을 구현한다. 게임은 아래와 같은 규칙으로 진행한다.

```
- 3개의 디스크와 막대가 존재한다 (디스크의 개수와 막대의 개수는 변경 가능하다.)
- 첫번째 막대에 3개의 디스크가 크기 순서대로 꽃혀있다.
- 3개의 디스크를 첫번째 막대에서 마지막 막대로 옮기면 종료된다.

- 한번에 1개의 디스크만 옮길 수 있다.
- 각 막대에서 가장 위의 디스크만 옮길 수 있다.
- 작은 디스크 위에 큰 디스크를 올릴 수 없다.

- 막대는 적어도 2개 이상이어야 디스크를 옮길 수 있다.
- 디스크가 막대 개수보다 많다면 게임이 종료될 수 없다.
```


### 입출력 요구 사항

#### 입력

- 막대와 디스크의 개수를 입력받는다. 단 막대의 개수가 2개 이하일 경우 예외처리한다.

```
3, 3
```

- 어느 막대의 디스크를 어느 막대로 옮길 것인지 입력받는다. 단 게임 규칙 위반의 경우 예외처리 및 재입력한다.

```
2, 1
```

- 게임 종료 후 재시작과 종료 여부를 입력받는다. 단 예외 값 입력 시 재입력한다.

```
Y
```


#### 출력

- 막대와 디스크의 개수 입력을 위한 요구문을 출력한다.

```
하노이의 탑 게임을 시작합니다.
막대와 디스크의 개수를 입력해주세요. 
```

- 현재 막대와 디스크 상태를 시각화 한다.

```
[1] 3 2 1
[2] 
[3]
```

- 디스크 위치 변경값 입력을 위한 요구문을 출력한다 (이동 횟수를 []형식으로 가장 앞에 출력한다)

```
[1] From which tower will you move a disk to which tower? (from = [1|2|3], to = [1|2|3]):
```

- 예외 상황 시 에러 문구를 출력해야 한다. 단, 에러 문구는 "=> Move failed!" 이다.
- 변경 성공 시 "=> Move succeeded!"를 출력한다.

```
=> Move failed!
=> Move succeeded!
```

- 게임 종료 시 종료 알림과 재시작 여부를 입력받는다.
```
Congratulation! You solved it in 7 moves!
Do you want to play again? (Y/S):
```
